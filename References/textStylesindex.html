CSS Fonts

Type of Fonts

  • Serif
  • Sans-serif
  • Monospace
  • Cursive
  • Fantasy

Serif fonts are best used for print media. Sans-serif fonts are best used for screen media. You can use Sans-serif fonts for a paragraph and serif fonts for headings.

Font-Size

  • xx-small, x-small, small, medium, large, x-large, xx-large
  • pixel
  • point
  • em
  • %

Font Properties

Font-Weight
The font-weight property sets how thick or thin the characters in text will display. Example: making something bold.
Font-Style
The font-style property specifies the font style for text. Example: italics or oblique.
Line-height
The line-height propery specifies the height of a line. This can be written as normal, 80%, or 1.6.
Text-align
The text-align property sets the alignment-- can be left aligned, centered, right aligned, or justified.
Text-decoration
Text decoration can be none, underline, overline, or line-through.
Text-indent
Text-indent property will specify the indentation of the first line in a text block.
Text-transform
Text-transform property will control the capitalization of text. Can be written as uppercase, lowercase, and capitalize.
Letter-spacing
Letter-spacing will increase or decrease the space between characters in text
